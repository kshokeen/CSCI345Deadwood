# Deadwood

## Building
Project can be built using:

.\gradlew.bat build

## Running
Start the game with 2 to 8 players:

.\gradlew.bat run --args="2"


Replace `2` with the number of players.

## Playing
On your turn, type one of the listed actions. Common actions are:

move
take role
act
rehearse
upgrade
state
help
end
quit

The game will show which actions are available for the current player.

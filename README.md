# Deadwood

## Building
Project can be built using:

.\gradlew.bat build

## Running
Start the game:

.\gradlew.bat run

The game will open a window and ask for the number of players.

## Playing
On your turn, click one of the listed actions. Common actions are:

move
take role
act
rehearse
upgrade
state
help
end
quit

The game will show which actions are available for the current player.
